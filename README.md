# Inventory Management System

A desktop inventory management application built with **C#**, **.NET**, and **Windows Forms**. This application allows users to manage inventory by creating, modifying, searching, and deleting parts and products through an intuitive graphical user interface.

> **Note:** This project was completed as part of the Bachelor of Science in Software Engineering program at Western Governors University and is shared to demonstrate software design, object-oriented programming, and desktop application development skills.

---

## Features

- Add, modify, and delete inventory parts
- Add, modify, and delete products
- Associate parts with products
- Search for parts and products by ID or name
- Input validation to prevent invalid data entry
- User-friendly Windows Forms interface
- Real-time inventory updates
